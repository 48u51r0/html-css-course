Headings Are Block Elements

Paragraphs are blocks, too. However, <em> and <strong> elements are not. They are inline elements.

Block elements define the flow of the HTML document, while inline elements do not.

Summary

Every HTML element is a box. Boxes come in two flavours:

  • Block Boxes
  • Inline Boxes

Boxes are defined by 4 properties:

  • Content: It's our text or media that we place in the element. It's the only piece with semantic value, That's why it is in the html.
  • Padding: Space between the box content and its border
  • Border: Line between the box's padding and margin
  • Margin: Space between the box and the surrounding boxes

Vertical Margins collapse

Two boxes one of top of the other have their vertical margins collapsed. That is to say, the distance between both will be the biggest vertical margin. They are not addtive.

When we want to bypass this working by de fault, we'll need to add an element between them.
